### &#x09;		**Types of Linked Lists**



###### **Singly Linked List**



1. A Singly Linked List consists of nodes where each node contains:
2. Data
3. Address of the next node
4. Traversal is possible only in the forward direction.
5. The last node points to NULL.



###### **Advantages:**

* Requires less memory.
* Easy to implement.



###### **Disadvantage:**

* Cannot move backward.



##### 

###### **Doubly Linked List**



1. A Doubly Linked List consists of nodes where each node contains:
2. Address of previous node
3. Data
4. Address of next node
5. Traversal is possible in both forward and backward directions.



###### **Advantages:**



* Can move forward and backward.
* Easier deletion and insertion operations.



###### **Disadvantage:**

* Requires extra memory for storing the previous address.
