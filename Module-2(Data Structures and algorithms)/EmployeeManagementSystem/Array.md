### &#x09;		       **ARRAY**



###### 

###### ARRAY REPRESENTATION IN MEMORY



1- Arrays store elements in continuous memory locations.

2- Each element occupies a fixed amount of memory based on its data type.

3- The first element is stored at the base address of the array.

4- The address of any element can be calculated using:

&#x09;Address = Base Address + (Index × Size of Data Type)



5- Because the memory locations are continuous, elements can be accessed directly using their index.

6- Arrays use indexing, where the first element is at index 0, the second at index 1, and so on.

###### 

###### ADVANTAGES



**Fast Access**

* Any element can be accessed directly using its index.
* Time Complexity: O(1)



**Easy Traversal**

* Elements can be visited sequentially using loops.



**Efficient Memory Usage**

* Arrays do not require extra memory for pointers or links.



**Simple Implementation**

* Easy to create and use in programming.



**Suitable for Fixed-Size Data**

* Useful when the number of elements is known beforehand.
